Introduction

Challenges

  • Existing DIR methods focus mainly on visual cues, limiting retrieval within coarse semantic classes
  • Difficult to capture rich and complex visual features in documents
  • Lack of fine-grained DIR datasets hinders supervised training
  • Insufficient evaluation on large-scale, real-world document images

Contributions

  • Propose the first fine-grained DIR benchmark in natural scenes
  • Conduct a comprehensive evaluation of mainstream cross-modal models
  • Develop a two-stage DIR framework with strong performance in real-world settings

Conclusion & Future Work

Summary

The Multimodal RAG System successfully achieved a Recall@5 of 68.33% and Faithfulness of 64.21%, demonstrating solid performance for general image retrieval with a latency of <100ms.

Key Achievements

  • Fast semantic search with FAISS
  • Minimal hallucination with RAG
  • Modern responsive web interface
  • Comprehensive evaluation framework

Future Directions

  • Local models for full privacy
  • Larger dataset for scalability
  • Video retrieval capabilities
  • Advanced re-ranking algorithms
  • Multi-user authentication system

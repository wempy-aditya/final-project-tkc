Introduction

Challenges

  • Traditional keyword-based retrieval fails to capture semantic nuances of complex queries
  • Generative models (LLMs) often suffer from hallucination without factual grounding
  • Difficult to integrate visual search with content generation seamlessly
  • Need for real-time performance in multimodal retrieval systems

Contributions

  • Implemented end-to-end Multimodal RAG architecture integrating retrieval and generation
  • Achieved semantic search with <100ms latency using CLIP + FAISS
  • Developed generative augmentation with LLM and Stable Diffusion
  • Created modern web UI with Dark Mode and real-time metrics

Conclusion & Future Work

Summary

The Multimodal RAG System successfully achieved a Recall@5 of 68.33% and Faithfulness of 64.21%, demonstrating solid performance for general image retrieval with a latency of <100ms.

Key Achievements

  • Fast semantic search with FAISS
  • Minimal hallucination with RAG
  • Modern responsive web interface
  • Comprehensive evaluation framework

Future Directions

  • Local models for full privacy
  • Larger dataset for scalability
  • Video retrieval capabilities
  • Advanced re-ranking algorithms
